Comprehensive exploratory data analysis on the red wine quality dataset, investigating factors that influence wine quality through statistical analysis and visualization.

- **Statistical Analysis**: Applied statistical methods to identify significant relationships between variables
- **Visualization**: Created informative visualizations to communicate findings effectively
- **Insights**: Derived actionable insights about factors affecting wine quality

### Technologies Used

- R Programming
- Statistical Analysis Libraries
- Data Visualization (ggplot2)
