## Project Overview

This deep learning project focuses on sentiment analysis for the Khmer language using Transformer architecture.

### Key Contributions

- **Attention Mechanism Analysis**: Broke down the concept of the attention mechanism and its properties, explaining step-by-step the backbone of the Generative Pre-trained Transformer (GPT)
- **Literature Review**: Conducted comprehensive review of Transformer architecture, broadening knowledge around the topic

### Technologies Used

- Python
- Transformer Architecture
- PyTorch
