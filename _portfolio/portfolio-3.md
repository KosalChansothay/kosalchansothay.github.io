## Project Overview

Implemented object detection and localization using the YOLOv8 architecture with comprehensive literature review on the mathematical foundations.

### Key Contributions

- **Model Implementation**: Initialized pre-trained YOLOv8 model for training and evaluation
- **Literature Review**: Conducted comprehensive review of the mathematical formula behind YOLO architecture
- **Results Interpretation**: Interpreted results from training/inference and created demo

### Technologies Used

- Python
- YOLOv8 (Ultralytics)
- Computer Vision Libraries
