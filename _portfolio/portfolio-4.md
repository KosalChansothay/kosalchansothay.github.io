## Project Overview

Conducted research investigating how coffee consumption affects university students' academic performance before examinations compared to non-consumer students.

### Key Contributions

- **Research Design**: Designed study to compare academic performance between coffee consumers and non-consumers
- **Exploratory Data Analysis**: Used EDA techniques to investigate data prior to model specification and diagnostics
- **Statistical Modeling**: Applied appropriate statistical models for hypothesis testing
- **Recommendations**: Interpreted research results and provided evidence-based recommendations

### Technologies Used

- R Programming
- Statistical Analysis Libraries
- Data Visualization
